## プロジェクト概要

- **言語**: Python
- **ゲームライブラリ**: Pyxel
- **内容**: Pyxelを使ったレトロスタイルのゲーム制作

## Pyxel

### 実行方法

```bash
python game.py
```

### 基本ルール

- スクリーンサイズはコンテンツから逆算して決める（先に固定サイズを決めない）
- `draw()` の先頭では必ず `pyxel.cls(col)` を呼ぶ
- 一度きりのアクションには `btnp()`、長押しには `btn()` を使う
- リストを走査しながら削除する場合はコピーを使う: `for e in list(enemies):`
- `sin()` / `cos()` は度数法（ラジアンではない）

### オーディオ

- BGMはch0〜2、SEはch3を使う（チャンネルの競合を避ける）
- SEの音量は5〜7（BGMより大きく、埋もれないように）

### スプライト

- プレイヤーは最低5枚（idle/walk×2/jump/attack）
- 敵は最低2フレームのアニメーション
- 透明色は `colkey=0`（黒を透明扱い）

### カラーパレット

```
0:黒  1:紺  2:紫  3:緑  4:茶  5:紺青  6:水色  7:白
8:赤  9:橙  10:黄  11:黄緑  12:シアン  13:灰  14:ピンク  15:肌色
```

- 背景（暗色）: 0, 1, 5
- 地形・環境（中間色）: 3, 4, 13
- プレイヤー・アイテム（明色）: 8, 10, 11
