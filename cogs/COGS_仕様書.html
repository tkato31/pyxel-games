4. 歯車オブジェクト

Gear クラス

{
    "x": int,          # 中心X座標
    "y": int,          # 中心Y座標
    "teeth": int,      # 歯の数
    "radius": int,     # 本体の半径（px）
    "angle": float,    # 現在の回転角度（度数法）
    "speed": float,    # 回転速度（度/フレーム）
    "fixed": bool,     # True=配置済み
    "tooth_len": int,  # 歯の長さ = max(3, radius // 3)
}

描画構造

  1. 歯（台形） — 各歯を2つの三角形で描画
  2. 本体（円） — 塗りつぶし円
  3. スポーク（4本） — 中心から歯の先端まで貫通する線
  4. 中心穴 — 背景色の円 ＋ リング ＋ 中心点

歯の形状パラメータ

base_half = pi / teeth * 0.7   (歯の根元の半幅)
tip_half  = pi / teeth * 0.4   (歯の先端の半幅)

5. 噛み合い判定

メッシュ距離

def mesh_dist(r1, r2):
    tl1 = max(3, r1 // 3)
    tl2 = max(3, r2 // 3)
    return r1 + r2 + max(tl1, tl2) + 2

速度比

speed_b = -speed_a * (teeth_a / teeth_b)

隣接する歯車は必ず逆方向に回転。歯数の比率で速度が決まる。

位相同期

def calc_mesh_angle(driver, driven_teeth):
    step = 360.0 / driven_teeth
    return driver.angle * (-driver.teeth / driven_teeth) + step / 2

隣接判定

配置済み歯車からの距離が outer_radius + 60 以内の最近傍を探索。隣が見つからない場合は配置不可。

操作フロー

  1. 手持ちエリアから歯車をクリックして選択（黄色ハイライト）
  2. 空きスロット（?マーク点滅）をクリックして配置を試みる
  3. 歯数が正解 → 配置成功（「カチッ」SE、回転開始）
  4. 歯数が不正解 → 「MISS!」表示（赤文字、ガコンSE）
  5. 全スロット配置完了 → クリア演出
  6. ENTERで次ステージへ

クリア演出

  • 全歯車の回転速度が徐々に加速
  • 歯車の色が4フレームごとに黄色と通常色で点滅
  • 120フレーム後に「PRESS ENTER FOR NEXT」表示

8. Phase 2 詳細設計

8.1 高解像度化（企画提案）

目的：ドットのガタつきを解消し、よりリアルで美しい歯車を描画する

アプローチ案：

  • 解像度を 320x240 → 480x360 または 640x480 に拡大
  • 歯車の半径も比例して拡大（描画ドット数増加で滑らかに）
  • 歯の形状パラメータの微調整
  • アンチエイリアス的な中間色による輪郭のなめらかさ向上
  • スポークの太さ（複数ライン描画）の検討

注意点：

  • Pyxelの最大画面サイズ制約の確認
  • ウィンドウ表示サイズとのバランス
  • 歯車サイズ変更時は mesh_dist により自動でスロット位置が再計算される

8.2 配置成功時の発光演出

配置直後の10〜15フレーム、歯車を明るい色（黄色）で描画。フレームカウンタで制御。

8.3 ゴースト表示（配置プレビュー）

手持ち選択中にスロット上にマウスを近づけると、点滅で歯車のシルエットを表示。配置前にサイズ感を確認できる。

8.4 ステージ開始演出

ステージ読み込み時に「STAGE N」を画面中央に大きく表示（2秒間）。フェードアウト後にプレイ開始。

8.5 ステージ間トランジション

クリア後のENTER → 画面が暗転 → 次ステージがフェードイン。

8.6 やり直し機能

配置済み歯車を右クリックで手持ちに戻す。以降の連鎖歯車も同時に外れる。

8.7 マウス追従

選択中の歯車の小さなアイコンをマウスカーソル位置に点滅描画。

8.8 BGM

  • ch0: ベースリズム（低音の規則的パルス）
  • ch1: 機械音（金属的な中音域のループ）
  • ch2: アクセント（高音域の短いフレーズ）

工場の機械が稼働している雰囲気を目指す。
