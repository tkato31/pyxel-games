## 1. ゲーム概要

### コンセプト

モーターの動力を、歯車をつないで、すべての機械に必要な分だけ届けるパズルゲーム。

### パズルの核心

1. **動力分散** — 分岐すると動力が均等に分かれる
2. **回転方向** — 歯車チェーンで方向が反転。機械は特定の方向を要求
3. **増幅と減衰** — 増幅ギア(+1)で動力を補う。減速ギア(-0.5)
4. **リソース制限** — 歯車の種類と数が限られている

### プレイヤーが考えること

- どこで分岐させるか（動力の配分）
- 増幅ギアをどの経路に入れるか
- 回転方向は合うか（奇偶数の制御）

## 4. ゲームメカニクス

### 4.1 動力伝達

- 歯車は上下左右に隣接する歯車と噛み合う（斜めは不可）
- モーターからBFS探索で連鎖的に動力が伝わる
- 孤立した歯車は回転しない

### 4.2 動力分散（v2の核心）

```
モーター(P4) → ギア(P4) → 分岐点(P4)
                              ├→ ギア(P2)  ← 動力が半分に！
                              └→ ギア(P2)  ← 動力が半分に！
```

- 分岐点から**N方向**に分かれると、各方向に**動力/N**が配分される
- 3方向分岐なら各1/3
- 増幅ギアで+1して動力を補える

### 4.3 増幅の加算式

```
P1 → 増幅ギア → P2（+1）
P0.5 → 増幅ギア → P1.5（+1）
```

- 増幅は**掛け算ではなく足し算**
- 入力値によって増幅後の値が変わる
- 「分岐前に増幅」と「分岐後に増幅」で結果が異なる → **位置が重要**

### 4.4 回転方向

```
モーター(↻) → 通常(↺) → 通常(↻) → 通常(↺) ...
            → 同方向(↻) → 通常(↺) ...
```

- 通常・増幅・減速・大型: 方向**逆転**
- 同方向: 方向**維持**
- 機械は特定の方向を要求（矢印アニメーションで表示）

### 4.5 配置ルール

- HandGearから歯車を選択し、グリッドの空きマスに左クリックで配置
- 配置するとアクションを1消費
- 右クリックで配置済み歯車をHandGearに戻す（アクション非消費）
- 固定ギアは右クリックで外せない

### 4.6 クリア条件

- 全ての機械が稼働状態であること
- 稼働 = 隣接歯車の回転方向が正しく、受信動力が要求値以上

### 4.7 動力表示

- 各歯車の下に動力値を「P数値」で表示
- 機械の下に必要動力値を表示（足りないと赤、足りると緑）

## 7. ヘルプ画面（4ページ）

タイトル画面で「H」キーを押すと表示。クリックまたは→キーで次ページ、←キーで前ページ。

### ページ1: 遊び方

```
遊び方

モーターから機械まで歯車をつないで動力を届けよう。
十分な動力を正しい回転方向で届けると機械が稼働する。

操作:
  左クリック : 歯車を選択 / 配置
  右クリック : 配置した歯車を外す
  Q          : 終了

目標:
  すべての機械を緑色にすること。
  赤 = 動力不足 または 回転方向が違う。
```

### ページ2: ギアの種類

```
ギアの種類

通常ギア（灰色）
  回転方向を逆転させる。動力はそのまま。

同方向ギア = （シアン）
  回転方向を維持する。動力はそのまま。
  方向の調整に使う。

増幅ギア + （橙色）
  回転方向を逆転させる。動力を +1.0 する。
  分岐で失った動力を補える。

減速ギア - （緑色）
  回転方向を逆転させる。動力を -0.5 する。

大型ギア L （水色）
  2x2マスを占有する。回転方向を逆転させる。
  多くの隣接セルを持つため分岐の起点に有利。
```

### ページ3: 動力の分岐

```
動力の分岐

歯車が2つ以上の歯車に接続すると、
動力が均等に分かれる！

  モーター(P4) → ギア(P4) → 分岐
     ├→ ギアA は P2 を受け取る
     └→ ギアB は P2 を受け取る

増幅ギアの位置が重要！

  分岐前に増幅: P1 → 増幅(P2) → 分岐 → 各P1
  分岐後に増幅: P1 → 分岐 → 各P0.5 → 増幅(P1.5)

  同じ増幅でも、置く場所で結果が変わる。
```

### ページ4: モーターと機械

```
モーターと機械

モーター（黄色）
  動力源。動力値（P1, P2...）を出力する。
  常に回転している。

機械（赤/緑）
  動力を届ける対象。
  矢印が必要な回転方向を示す。
  数字が必要な動力値を示す。
  条件を満たすと赤→緑に変化。

デュアル機械（三重枠）
  2つ以上の隣接歯車が回転している必要がある。
  すべての隣接歯車の動力が合算される。
```
